Privacy Policy

Last updated: June 29, 2026

1. Information We Collect

toko collects and stores the following information to operate the Service:

  • TikTok OAuth access tokens and refresh tokens (stored locally on the user's device)
  • TikTok account ID and posting history (stored in a private database for scheduling purposes)
  • Video file metadata (filename, upload timestamp)

2. How We Use Information

The information collected is used solely to:

  • Authenticate with TikTok's Content Posting API on behalf of the user
  • Schedule and publish videos to the user's TikTok account
  • Track posting history to prevent duplicate posts

3. TikTok API Data

This application uses TikTok's official API. Data accessed through the API is used only for the purposes described above and is not shared with any third parties. We comply with TikTok's API Terms of Service and data usage policies.

4. Data Storage

OAuth tokens are stored locally on the user's device. Posting records are stored in a private Supabase database accessible only to the account owner.

5. Data Sharing

We do not sell, trade, or share any user data with third parties.

6. Data Retention

Posting history is retained for operational purposes. Users may request deletion of their data at any time by contacting us.
